ImGuiWindowFlags_None

    Description: No flags set. The default state for a window.

ImGuiWindowFlags_NoTitleBar

    Description: Disables the title bar. The window will not have a title or be draggable by its title bar.

ImGuiWindowFlags_NoResize

    Description: Disables resizing the window.

ImGuiWindowFlags_NoMove

    Description: Disables moving the window by dragging it.

ImGuiWindowFlags_NoScrollbar

    Description: Disables the scrollbar. This will prevent the window from showing scrollbars, even if the content overflows.

ImGuiWindowFlags_NoScrollWithMouse

    Description: Disables scrolling with the mouse wheel. This will stop the window content from scrolling when the mouse is hovered over the window.

ImGuiWindowFlags_NoCollapse

    Description: Disables the collapse button, which hides the window content.

ImGuiWindowFlags_AlwaysAutoResize

    Description: Forces the window to automatically resize based on its contents. It will resize even if the window size is manually set.

ImGuiWindowFlags_NoBackground

    Description: Disables the background of the window, making it transparent.

ImGuiWindowFlags_NoSavedSettings

    Description: Prevents saving of the window’s position and size. It will always reset to the default size when reopened.

ImGuiWindowFlags_NoMouseInputs

    Description: Disables mouse interactions with the window (e.g., clicking, moving, resizing).

ImGuiWindowFlags_MenuBar

    Description: Enables a menu bar at the top of the window. If this is enabled, the ImGui::BeginMenuBar()/ImGui::EndMenuBar() will be used.

ImGuiWindowFlags_HorizontalScrollbar

    Description: Enables a horizontal scrollbar when the content exceeds the width of the window.

ImGuiWindowFlags_NoFocusOnAppearing

    Description: Prevents the window from gaining focus when it first appears.

ImGuiWindowFlags_NoBringToFrontOnFocus

    Description: Prevents the window from being brought to the front when it receives focus.

ImGuiWindowFlags_AlwaysVerticalScrollbar

    Description: Forces the vertical scrollbar to always be shown, regardless of whether it is needed.

ImGuiWindowFlags_AlwaysHorizontalScrollbar

    Description: Forces the horizontal scrollbar to always be shown, regardless of whether it is needed.

ImGuiWindowFlags_ChildWindow

    Description: Marks the window as a child window (to be used inside another window). The window will have its own clipping region and be inside the parent window.

ImGuiWindowFlags_Clipper

    Description: Forces the window to be clipped inside the parent window, preventing any content from overflowing the window boundaries.

ImGuiWindowFlags_NoFocusOnClick

    Description: Prevents the window from gaining focus when clicked (useful if you don’t want a window to gain focus when clicked).

ImGuiWindowFlags_AlwaysAutoResize

    Description: Ensures the window size always adjusts automatically to fit its contents. If you manually set the window size, this flag will override it.
